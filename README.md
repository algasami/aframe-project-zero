# AFrame Project Zero

A basic AFrame project for my VR/XR class. This project was bundled using Webpack to enable a higher degree of
modularity. It also incorporated the official AFrame inspector for runtime game-level design and debugging.

- Goal: Interpret the poem 馬致遠《天淨沙·秋思》 using AFrame.

## Contribution Guidelines

1. Models should always be processed in 3D software, preferably Blender, including those already in compatible formats.
   This allows optimization efforts such as mesh merging, bone merging and decimation operations.
